Windows Telemetry Assessment Tool

=============================================================================================================================



Firstly, ensure **.NET 10 Framework** is installed. The application will not run otherwise, an error message will come up when trying to run the tool without .NET installed.



To install .NET 10, navigate to this link and install the .NET 10.0.6 Runtime: https://dotnet.microsoft.com/en-us/download/dotnet/10.0





Then, to run the tool, follow these steps:



1\. Enter the 'Builds\App' Folder - by this README.txt file.

2\. Scroll down the folder to find the application TelemetryToolGUI.exe (this is the prototype to run).

3\. This application is best run in administrator mode, this is done by right clicking the application, and clicking 'Run as administrator'.

4\. If the tool is not run in admin mode, you will be greeted with a pop up reminding you so. The tool can still be used.

5\. The tool contains tabs at the top for each scan, adjust the window size to view the blue button at the bottom to initiate the scans.

6\. To exit the tool, press the red cross at the top right, or Alt + F4.

For more information regarding the nature of this project and tool, read through the .pdf report named FYP_Report :)
